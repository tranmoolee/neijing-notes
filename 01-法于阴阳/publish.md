## Part 1 · 文生图 Prompt(共 6 张)

> **风格延续上一期定下的基调**:写实摄影 + 自然光 + 文艺胶片质感。
> **新增一个视觉锚点**:这一篇的"鳗鱼"比喻,可以作为这一期独一无二的封面记忆点。

### Prompt 1 · 主封面 · 鳗鱼隐喻
> 用途:博客头图 / 小红书封面 / 系列第一篇专属视觉

```
一只手部特写,正试图握住一条银黑色的鳗鱼,
鳗鱼湿滑,身体一半在手中,一半正从指缝间滑出,
水珠飞溅,瞬间凝固;手指因用力略微紧绷。
背景为深色木质桌面,旁边模糊可见一本摊开的古籍一角。
风格:静物高速摄影,黑白主调,
仅鳗鱼皮肤的银光和水珠反光带有微弱冷色,
画面有戏剧张力但克制,National Geographic 摄影质感。
氛围:讽刺中带哲思,寓意"以为抓住了又滑掉"。
比例:3:4(小红书) / 16:9(博客头图)
```

> 备注:这张图是这一篇的核心视觉,小红书封面用它读者会停下来。
> 如果出图困难,备选方案见 Prompt 1B。

### Prompt 2 · 中午十二点的书桌
> 用途:文中"我用我自己举例子"段落

```
正午阳光透过窗户斜射进书房,
桌上有一本翻开的笔记本、一杯黑咖啡、
一台合上的银色笔记本电脑;
窗外可见模糊的西班牙地中海街景轮廓。
桌面右侧有一本中文古籍,书签夹在中间。
风格:生活场景摄影,自然光强烈但柔和,
色调温暖,空气中有可见的微小尘埃浮动,
氛围安静、思考、略带困惑。
比例:3:4
```

### Prompt 3 · 注家分歧
> 用途:文中"查了什么"段落

```
俯视一张木桌,桌面上摊开三本不同版本的中医古籍,
每本都翻在不同页面,书页边缘有便签纸标注;
中间一本笔记本上手写多种解释,有的被划掉,
旁边散落几张写有不同观点的小纸条;
一支铅笔斜搭在笔记本上。
风格:学术工作台摄影,自然窗光,
色调米白配暗木色,极简但信息密度高,
真实学习状态,无摆拍感。
比例:3:4 或 1:1
```

### Prompt 4 · 日落与节律
> 用途:文中"日落了你睡觉"段落 / 视觉转折点

```
一扇朝西的窗户,夕阳正落入地平线,
天空呈现橘红到淡紫的渐变;
窗台内侧放着一只白瓷茶杯,杯中茶水反射夕阳的金光;
房间内已经开始变暗,只有窗边一线暖光。
风格:风景人文摄影,黄金时刻光线,
色调温暖怀旧,Fujifilm Velvia 色彩,
氛围宁静,有"该结束一天了"的暗示。
比例:3:4
```

### Prompt 5 · 留坑的便签
> 用途:文中"留个坑"段落 / 系列符号

```
特写一本笔记本的右下角,贴着三张不同颜色的便签,
分别用钢笔手写:
"道是什么?——以后再说"
"术数 = ???"
"以酒为浆,留给下篇"
便签下压着一支钢笔,笔记本本体可见手写笔迹的边缘。
风格:文具静物摄影,自然侧光,
色调米黄温暖,极小景深,
真实手写感,有学习节奏的仪式感。
比例:1:1 或 3:4
```

### Prompt 6 · 字太认识
> 用途:文中结尾"金句"段落

```
特写一本古籍的内页,"法"、"于"、"阴"、"阳"
四个繁体字被红色铅笔分别圈出,
每个字旁边都写着"认识",
但页面下方用钢笔写着一行小字:
"合起来读不懂"。
风格:微距静物摄影,自然光,
极简构图,黑白文字配红色圈注,
富有反差感和幽默感。
比例:1:1
```

## Part 2 · 小红书图文版(9 图卡片)

> **这一期的小红书结构有意识地"前重后轻"**:
> 前 3 张密集输出"承认看不懂"的反差感,中间靠注家分歧拉学术深度,后 3 张落到"自我观察",节奏是 **冲突 → 探索 → 顿悟 → 沉淀**。
> 这种节奏在小红书图文里收藏率高于一般"信息流式"图文。

## 📐 这一期视觉的小心思

| 元素 | 用意 |
|---|---|
| 封面用"鳗鱼"而不是古籍 | **打破预期**——读者刷到一个抓鳗鱼的画面会停 0.5 秒,而停 0.5 秒就够了 |
| 卡 5 极短文字 | **节奏点**——9 张卡里需要一张"喘气卡",字越少冲击力越大 |
| 卡 6 用三本古籍并排 | **暗示研究感**——读者潜意识里会觉得"这博主真的在查" |
| 卡 8 用便签纸 | **视觉符号**——"留坑便签"以后可以做成系列标志,每篇结尾都用 |

## 🔁 关于 Prompt 1 的鳗鱼图

我必须提醒一句:**鳗鱼这种比喻视觉,AI 出图成功率不高**——
很多模型对"湿滑、滑落、动态瞬间"这类描述把握不准,
出来的图可能很别扭。

**建议先试 Prompt 1,如果三次以内出不了满意的,直接用 Prompt 1B**。
1B 也很扎实,只是没那么"破圈"。

如果你 Mac mini 上跑的是 Flux,鳗鱼那张大概率能成;
如果是 SDXL,1B 更稳;
如果用即梦/Midjourney,Prompt 1 我建议把"水珠飞溅,瞬间凝固"改成"motion blur, water splash"再试。
