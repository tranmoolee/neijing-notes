## 两种酸，酸得不一样

先说一件读这篇之前想不通的小事。

在西班牙这几年，走山路的机会不少。我早就注意到一个现象：不同的路线，腿酸的位置不一样。长距离下山，酸的是小腿**外侧靠前**那条棱；有一回沿着海边的软沙滩走了一下午，第二天酸的却是小腿**内侧**那一溜。

当时的结论是：腿要散架了，位置随机。

读完足六经的循行原文，我把裤腿撸起来对了一遍——那两个位置，一个在胃经的道上，一个在脾经的道上。酸的位置不随机，**小腿是分车道的。**

酸本身多半是肌肉的事，我分得清这个。让我发毛的是另一件事：古人画的线，跟我腿上"会酸的那几条沟"，位置能对上。

## 小腿的车道划分

手六经的规则搬过来，一模一样：

> 足之三阳，从头走足；足之三阴，从足走腹。

阳经从头往脚走，负责外侧和后面；阴经从脚往肚子走，负责内侧。小腿上排布得非常整齐：

**外侧和后面，三条阳经：**

| 位置 | 经 | 起点 → 终点 |
|---|---|---|
| 前缘（胫骨外侧那条棱） | 胃经 | 鼻旁 → 第二个脚趾 |
| 正外侧 | 胆经 | 外眼角 → 第四个脚趾 |
| 后面（小腿肚正中） | 膀胱经 | 内眼角 → 小脚趾外侧 |

**内侧，三条阴经：**

| 位置 | 经 | 起点 → 终点 |
|---|---|---|
| 靠前（贴着胫骨内缘） | 脾经 | 大脚趾内侧 → 胸腹 |
| 中间 | 肝经 | 大脚趾背 → 胁肋 |
| 靠后 | 肾经 | 小脚趾下·足心 → 胸腹 |

这个可以自己上手：手指从胫骨外侧那条棱按下去，是胃经的道；小腿肚正中往下捏，是膀胱经的道；贴着胫骨内缘往上推，是脾经的道。不用记穴位，先记得腿上有六条道就行。

（还是那句话：这是拿自己的腿核对地图的定位游戏。定位归定位，"按了能怎么样"不归这个系列管。）

## 六条里最震的一条：膀胱经

膀胱经的循行我读了三遍。

起点在**内眼角**。上额头，过头顶，下后脑，然后沿着脊柱两侧一路向下——夹着脊梁到腰，穿过臀部，顺大腿后面、小腿肚正中，绕到外踝后面，最后停在**小脚趾外侧**。

从眼睛到小脚趾。整个人的背面——后脑、脖子、整条背、整条腿的后侧——全是它一条经的地盘。它也是十二经里穴位最多的一条。

读到这儿，第三十八篇的一个疑问自动解开了。《热论》说外感病第一天"巨阳受之"，先病的是太阳经，症状是头项痛、腰脊强。当时我不懂为什么受寒先中这条。现在看图明白了：**太阳膀胱经就是人体的整个背面**——你迎风站着，或者夜里踢了被子，最先暴露给寒气的那一大片，全是它。头项痛、腰脊强，就是这条线从上到下在报警。

还有个日常对得上的：落枕、久坐后那种从后脑一直僵到后腰的感觉，走的正是这条线的路径。位置对上了，原因我不下结论。

## 另外两条的惊奇点

**胃经**：从鼻旁出发，绕过脸（所以它是阳经里唯一走**身体正面**的），下颈过胸，穿过整个肚子，沿大腿前面、胫骨外侧那条棱，一路到第二个脚趾。从脸到脚，纵贯全身正面——难怪主线里脾胃被叫作"后天之本"，这条运粮干道是真的长。

**肾经**：它的起点是十二经里唯一一个在**脚底**的。"起于小指之下，邪走足心"——从小脚趾下面斜着穿到足心（涌泉那个位置），再绕内踝上行。全身的经，只有肾经是从踩地的那一面发源的。这个安排我越想越有意思，先放着。

## 到这篇，圈闭上了

上一篇说手上六条经是"三组接力赛"，阴经在指尖把棒交给阳经。这一篇把足六经拼进去，我发现交接棒有一套完整的规则：

- **阴经交阳经，在手脚末端**（肺→大肠在手指，脾→胃反向衔接同理）
- **阳经交阳经，在头面**（小肠经到内眼角结束——膀胱经就从内眼角出发；大肠经到鼻旁结束——胃经就从鼻旁出发）
- **阴经交阴经，在胸腹**（脾经注入心中，交给心经）

十二条经首尾相焊，一条不断：

肺 → 大肠 → 胃 → 脾 → 心 → 小肠 → 膀胱 → 肾 → 心包 → 三焦 → 胆 → 肝 → 回到肺。

第四十一篇总纲里"如环无端"四个字，到这儿落实了。经脉不是十二条各自的线，是**一条折返十二次的环线**，手三次、脚三次、来回穿梭，把全身织在一张网里。

至于气在这条环线上按什么节奏跑、一天跑多少圈——原文有个说法叫"五十而复大会"，留给第四十六篇《营卫生会》去读。

## 这一段暂时的理解

用大白话复述（依然标注"未必对"）：

> 足六经和手六经同一套交通规则：三条阳经从头走脚（胃经管正面、胆经管侧面、膀胱经管整个背面），三条阴经从脚走腹（脾、肝、肾分占小腿内侧三条道）——在自己小腿上就能挨个摸出位置。膀胱经从内眼角铺到小脚趾，独占人体背面，这解释了第三十八篇"伤寒一日巨阳受之"为什么受寒先中它。而且交接棒有规则：阴阳交接在手脚末端、阳阳在头面、阴阴在胸腹——十二条经由此焊成一个闭环，"如环无端"不是修辞，是拓扑结构。

## 留的坑

1. **环线为什么从肺开始？** 流注顺序肺→大肠→胃→脾……这个起点是怎么定的？跟"谷入于胃"、跟呼吸有关系吗？第四十六篇读营卫时再看。
2. **经和穴是什么关系？** 线画完了，可线上那些"点"（穴位）是怎么定出来的？下一篇《本输》正面回答。
3. **酸的位置和经的位置重合，到底说明什么？** 肌肉解剖也能解释酸的分区（下山伤胫骨前肌，沙地用内侧稳定肌）。两套解释各管各的，还是同一件事的两种语言？老实说我分不清，记下。
4. **肝脾两条线在小腿上还换过一次道**（原文"交出太阴之后"）——内踝上八寸处两条经前后互换位置。为什么要设计这个交叉？没查到让我信服的说法。
