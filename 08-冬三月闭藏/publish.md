### Prompt 5 · 卡 5 · 闭环

**生图描述：**
```
俯视角，桌面上用四片不同颜色的叶子
排成一个圆环/循环：
嫩绿色新芽（春）→ 深绿色大叶（夏）
→ 金黄色枯叶（秋）→ 棕色干叶碎片（冬）；
四片叶子之间用手绘的箭头连接成循环；
白色桌面，自然光，干净极简。
风格：俯视静物摄影，自然窗光，
高调明亮，有手工感和图解感。
比例：1:1
```

**叠加文字：**
```
主标题：
四季闭环合上了

正文：
春发 → 夏长 → 秋收 → 冬藏 → 春发……

每一季的产出
是下一季的输入

冬藏不够 → 春发不了
春发不了 → 夏长不起来
→ 秋收不到 → 冬更没东西藏

小字：
不是四条独立的建议
是一个正反馈系统
```

```
《学渣读内经》第八篇

四气调神大论，最后一季——冬天。

冬天的核心三个字："无扰乎阳"。
阳气已经藏起来了，别去打扰它。
太阳没出来别起。
不是懒，是配合。

但这一篇最打动我的不是养生动作，
是一句描写内心状态的话：

"若有私意，若已有得"

不追、不缺、不展示。

我的日常是永远在追下一个，
永远觉得还差一点。
这句话描述的状态，我太陌生了。

古人说冬天应该活在这种状态里。
不知道算不算养生，
但就算不算，我也想试试。
```
