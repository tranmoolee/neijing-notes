# 《学渣读内经·序》—— 配图 Prompt + 小红书图文版

## Part 1 · 文生图 Prompt(共 6 张,可按需挑用)

> 风格统一原则:**写实摄影 / 自然光 / 文艺胶片质感 / 温暖克制 / 避免 AI 感和过度装饰**。
> 这套调性和你已发的两篇旧文(诊室、针灸、传承叙事)是一脉相承的,小红书读者跳过来看博客也不会错位。

### Prompt 1 · 主封面 · 书桌学习场景
> 用途:博客头图 / 小红书封面 / 系列总封面

```
一张深色木质书桌的俯视角,中央摊开一本线装《黄帝内经》古籍,
泛黄宣纸,繁体竖排;旁边一本横线笔记本打开,
上有钢笔手写的中医笔记(可见划掉修改的痕迹);
一支毛笔斜搭在笔架上,右上角一杯热茶冒着淡淡热气,
左下角随意散落几枚银色针灸针。
下午斜射的暖黄窗光,真实木桌纹理,景深浅。
风格:写实摄影,Kodak Portra 400 胶片色彩,文艺克制,
画面留白干净,生活气息真实,避免商业广告感。
比例:3:4(小红书) / 16:9(博客头图)
```

### Prompt 2 · 父亲针灸场景 · 传承特写
> 用途:文中"我爸做针灸"段落 / 小红书第 2 张

```
中近景特写,只露出一双有岁月感的中年男性手,
正在拈起一枚约 4cm 长的银色针灸针,准备进针。
背景大幅虚化,隐约可见白色诊疗床一角和淡黄色窗光。
手部皮肤纹理真实,光线柔和,情绪克制。
风格:写实人像摄影,匠人气质,故事感强,
色调暖灰,Leica 镜头质感,
避免任何医疗广告或戏剧化打光。
比例:3:4
```

### Prompt 3 · 文化交错 · 西班牙窗台
> 用途:文中"移民之后"段落 / 小红书第 3 张

```
欧式公寓飘窗台,窗外可见模糊的西班牙街景
(米色石材建筑、木质百叶窗、强烈地中海日光);
窗台内侧摆放东方物件:一只白瓷茶杯、一本中文古籍、
一小盒针灸针,旁边随意放着一张西班牙语血液检查报告
(个人信息模糊不可辨);窗内窗外形成文化错位感。
风格:生活场景摄影,自然光,色彩明亮但不饱和,
安静日常的氛围,Fujifilm 色彩。
比例:3:4
```

### Prompt 4 · 数据与笔记 · 学习仪式感
> 用途:文中"复查报告"段落 / 小红书第 4-5 张

```
俯视一张工作桌:左侧一份西班牙语血常规检查报告
(细节模糊处理,只可见表格结构和上下箭头);
中间一本打开的笔记本,钢笔手写"Hb 6.20 → 7.80"、
"脾不统血,气不摄血"等关键词;
右侧一本《黄帝内经》压在角落,
旁边一支钢笔和一杯半凉的茶。
风格:扁平俯视摄影,极简构图,
自然窗光,知识工作者氛围,
色调偏米白与原木色,真实纸张纹理。
比例:3:4 或 1:1
```

### Prompt 5 · 古籍特写 · 经典感
> 用途:文中"为什么从《内经》开始"段落 / 小红书第 7 张

```
特写一本线装《黄帝内经》的内页,
黑色繁体竖排字体清晰可见,纸张为略发黄的宣纸质感;
书页右下角夹一枚素色棉布书签,
一支红色铅笔斜放在页面上,似刚做完批注。
背景为深色木桌一角。
风格:静物特写摄影,中式文化质感,
光线柔和侧逆光,色调米黄,
纸张纹理细腻,Hasselblad 中画幅质感。
比例:1:1 或 4:5
```

### Prompt 6 · 学渣场景 · 真实学习状态
> 用途:系列调性图 / 小红书第 8 张("四条规矩"那张)

```
一张稍显凌乱的书桌,一摞中医书堆在角落
(可见《黄帝内经》《伤寒论》《本草纲目》书脊);
中间一本打开的笔记本,大量手写、划掉、再写的痕迹;
旁边一杯凉透的茶,一支咬过笔头的铅笔;
书页右下角贴一张便签,上面写着"看不懂,先放一放"。
风格:真实生活摄影,自然光,
有点凌乱但温暖,有自嘲式的幽默感,
色调暖中带一点冷,避免摆拍感。
比例:3:4
```

## Part 2 · 小红书图文版(9 图卡片 + 正文 + 标签)

> **小红书的核心规则**:封面定生死,前 3 张卡决定完播率,大字标题要能在缩略图状态下看清。
> **文字密度**:每张卡正文不超过 60 字,关键词加粗或单独成行。
> **图片建议**:封面用 Prompt 1,内页可混用 Prompt 2-6,文字直接叠加在图上(用浅色半透明色块衬底以保证可读性)。

### 📝 小红书发布正文(配在 9 图下方)

```
《学渣读内经》系列开更 📖

我爸做针灸几十年,我自己一直只是
"耳濡目染、会一点,但没真学"。

直到在西班牙帮一个朋友的崩漏问题
做了几次针灸,几个月没停的出血止住了。
前两天她复查血常规,Hb 从 6.20 升到 7.80。

这件事让我决定:
是时候老老实实从地基开始学了。

地基,就是《黄帝内经》。

我不是中医,不是医生,
不会给任何人远程开方诊断。
我只是一个想认真读经典的学习者。

会读错。会留坑。会回来打自己脸。
欢迎一起读,也欢迎指正 🙏

完整版在博客(主页链接 / 评论区置顶)
```

## 📐 排版建议(图卡制作时)

| 元素 | 建议 |
|---|---|
| 字体 | 标题:思源宋体 / 霞鹜文楷;正文:苹方 / 思源黑体 |
| 主色 | 米白底 + 墨黑字,点缀朱砂红(印章感) |
| 标题位置 | 居中或左上,留 1/3 画面 |
| 行距 | 正文 1.6-1.8 倍,呼吸感很重要 |
| 工具 | 推荐稿定设计 / Canva / Figma 模板批量化 |

如果你那台 Mac mini 上跑了本地图像模型,这套 prompt 可以直接喂给 SDXL/Flux 系列;
如果用 Midjourney/即梦,把"风格"那段裁掉、保留场景描述效果更好(它们对风格关键词反应更敏感)。
