# 第二阶段回头看 —— 配图 Prompt + 小红书图文版

> 配图工作流（第一原则）：只用 image-2 生成**无字**国风画面，构图上方/一侧留出深色负空间；中文标题一律后期排版叠加（见各卡「叠加文字」）。除古籍书页上的文字外，画面不出现任何文字 / 标注 / 水印。

---

## 风格后缀（所有 Prompt 统一追加）

```
暖调国风静物摄影，自然光 / 侧逆光，胶片质感（Kodak Portra 400），
米褐与墨色为主，留大面积负空间用于后期叠字，
构图安静、克制、有呼吸感，无文字、无标注、无水印。3:4 竖构图。
```

## Part 2 · 小红书图文版（9 图卡片）

### 封面卡

**叠加文字:**
```
那张"五行大表"
根本不是用来背的

学渣读内经
第二阶段回头看 · 10-17
```

### 卡 1 · 两本不相干的书

**正文:**
第二阶段跨了两本经文，画风差到离谱——

生气通天论：讲道理
"别糟践自己"

金匮真言论：列表格
"东方青色入通于肝…"

一个温情，一个枯燥
我一度以为编错了顺序

### 卡 2 · 上半场一句话：自伤

**正文:**
生气通天论那五篇，压成一句——

病，不是天降的，是自伤

「虽有贼邪，弗能害也」
害得了你的，是
「卫气散解，此谓自伤」

整篇没有一味药
全是"别"

### 卡 3 · 下半场：给五脏画地图

**正文:**
翻到金匮真言论，画风突变
古人开始干一件理工科的事——

给五脏建坐标系

方位×季节×脏×部位
再加颜色、味道、声音、数字…
一张越画越细的地图

### 卡 4 · 我第一反应：完蛋要背

**正文:**
读到那张九行大表时
我生理性抗拒——

肝青心赤脾黄肺白肾黑
肝酸心苦脾甘肺辛肾咸
肝八心七脾五肺九肾六…

又是一张要死记硬背的表？
记住"肝的数是八"对我有啥用？

### 卡 5 · 转折：它是一台定位仪

**正文:**
让我转过弯的，是一个字：俞

"病在肝，俞在颈项"

记它不是为了考试，是为了反查——
颜色不对 → 定位到脏
总想吃某个味 → 定位到脏
老在某季节犯病 → 定位到脏

表越细，不是负担，是精度

### 卡 6 · 坐标 × 自伤 = 报警地图

**正文:**
两本书"啪"地合上了——

上半场：病是自伤（为什么报警）
下半场：坐标定位（在哪报警）

只看上半场：知道坏了，不知坏在哪
只看下半场：有表，不知它指向什么

缝起来，就是一张报警地图

### 卡 7 · 阴中有阴：给地图调焦

**正文:**
最玄的"阴中有阴"，其实最实用——

白天还能分上午（阳中之阳）
和下午（阳中之阴）
分了还能再分

它在告诉你：这张地图
分辨率可以无限放大
刻度越密，定位越准

### 卡 8 · 下期预告

**正文:**
第二阶段教我一句话：
先认账，再定位

不舒服别急着往外甩锅
先问是不是自伤
再读身体给的信号

下一阶段，最硬的一篇
《阴阳应象大论》
我做好了被虐的准备

## 排版说明

| 元素 | 字号建议 |
|---|---|
| 卡片标题 | 36-40pt，加粗 |
| 卡片正文 | 24-28pt，常规 |
| 封面主标题 | 46pt，加粗 |
| 封面副标题 | 24pt，常规 |
| 系列标识 | 18pt，置于角落 |

- 配色：暖棕 + 米白，延续系列视觉风格
- 图片用 image-2 生成**无字**画面后，按各卡「叠加文字」把标题后期排版叠上去
- 封面卡需确保缩略图（200x267px）下主标题可读
- 卡 6（核心金句卡）建议居中排版，留白多一些
