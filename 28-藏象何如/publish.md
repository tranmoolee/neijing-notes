## Part 2 · 小红书图文版（9 图卡片）

### 封面卡
**叠加文字:**
```
藏象何如

内脏明明藏在身体里
却一直把状态
挂在外面

学渣读内经 · 第二十八篇
```

### 卡 1 · 钩子
**标题:** "藏象"这个词，本身就是答案
**正文:**
藏 = 藏在体内、看不见的脏
象 = 显在外、看得见的征象

藏在里面的东西
会以"象"显现到外面

内脏一直挂在外面
你只是没学会看

### 卡 2 · 五块仪表盘
**标题:** 每个脏的"仪表盘"在哪
**正文:**
心 → 华在面（脸色）
肺 → 华在毛（皮肤光泽）
肾 → 华在发（头发）
肝 → 华在爪（指甲）
脾胃 → 华在唇（嘴唇）

里面的脏
写给你的报告
就贴在这些地方

### 卡 3 · 你天天在读，只是没意识到
**标题:** 原来我们一直在读内脏
**正文:**
看脸色好不好 → 在读心
看头发枯不枯、白不白 → 在读肾
看指甲有没有竖纹 → 在读肝
看嘴唇淡不淡 → 在读脾胃
看皮肤润不润 → 在读肺

外面的象
是里面的脏写的报告

### 卡 4 · 和第三篇接上了
**标题:** 难怪古人盯着头发讲衰老
**正文:**
第三篇"女七男八"
讲衰老反复说：发白、发堕

不是随便举例——
头发是肾的"外显指针"
肾精盈亏会写在头发上

不是头发在变
是它背后那块"表"在变

### 卡 5 · 还连着季节
**标题:** 每个脏，各通一个季节
**正文:**
心 通于夏气
肺 通于秋气
肾 通于冬气
肝 通于春气
脾胃 通于土气

五脏不只连着体表
还连着四时
（呼应第十六篇那张大表）

### 卡 6 · 一颗没拔的钉子
**标题:** 凡十一脏取决于胆也
**正文:**
心是君主、肺是宰相
怎么最后说
十一个脏都"取决于胆"？

我的临时理解（不一定准）：
胆主决断
再全的系统
最后总得有人拍板

### 卡 7 · 对普通人有什么用
**标题:** 不用体检，先学会"看表"
**正文:**
最近脸色暗、嘴唇淡？
头发掉得多、指甲起纹？

先别慌、也别自我诊断
但可以把它当成
身体发来的"报告"

看见信号
比忽略信号好

（提醒：异常持续，请就医）

### 卡 8 · 下期预告
**标题:** 下一篇：五脏不是五个孤岛

### 小红书发布正文
```
这一篇刷新了我对"内脏"的认知。

"藏象"两个字本身就是答案：藏（看不见的脏）+ 象（显在外的征象）。
内脏虽然藏在体内，却一直把状态挂在外面——心写在脸、肺写在皮毛、肾写在头发、肝写在指甲、脾胃写在嘴唇。

所以看脸色、头发、指甲，本质是在读内脏的"仪表盘"。
（也终于明白第三篇为什么盯着头发讲衰老：发是肾的指针。）

每个脏还各通一个季节：心通夏、肺通秋、肾通冬、肝通春。
最后一句"凡十一脏取决于胆"我还没真懂，先存疑。

对普通人的用法：把脸色/头发/指甲的变化当成身体发来的"报告"，看见信号——异常持续就去看医生。

详细笔记见主页博客链接~
```
